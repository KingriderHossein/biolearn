# BioLearn 🧬
پلتفرم آموزش ویدیویی زیست‌شناسی

## راه‌اندازی
1. ریپازیتوری را در گیت‌هاب ایجاد کنید.
2. فایل‌ها را push کنید.
3. در تنظیمات GitHub Pages دامنه‌ی biolearn.ir را تنظیم کنید.
4. فایل CNAME را در ریشه قرار دهید.

## ساختار فایل‌ها
- `index.html` : صفحه اصلی
- `css/style.css` : تمام استایل‌ها
- `js/main.js` : عملکرد جستجو و فیلتر
- `pages/` : صفحات هر بخش زیست‌شناسی

## ویرایش ویدیوها
در هر فایل بخش (مثلاً `cell-molecular.html`)، کامنت `<!-- iframe آپارات -->` را با لینک embed خود جایگزین کنید.
